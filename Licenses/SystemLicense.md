# System License

Defines the license scope for the entire system installation.

## Entitlements

| Property | Description |
| --- | --- |
| `SystemId` | Unique identifier for the system (ULID). |
| `EnvironmentName` | The deployment environment name (e.g., Production, Staging). |
| `MaximumCompanyCount` | The maximum number of tenant companies allowed in the system. |
| `Countries` | List of supported countries (ISO region codes) where the system is authorized to operate. |
