# Company License

Defines the license scope for a specific tenant company within the system.

## Entitlements

| Property | Description |
| --- | --- |
| `SystemId` | Reference to the System ID. |
| `CompanyId` | Unique identifier for the company. |
| `MaximumBranchCount` | The maximum number of branches this company can create. |
| `MaximumEmployeeCount` | The maximum number of employees this company can manage. |
| `MaximumDepartmentCount` | The maximum number of departments this company can define. |
